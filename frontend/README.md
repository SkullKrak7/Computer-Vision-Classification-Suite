# Frontend - React Web Interface

**Built with Kiro CLI** - React components developed using Amazon's AI-powered development assistant.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Visit http://localhost:3000

## Build

```bash
npm run build
```

## Features

- Live inference with image upload
- Training progress monitoring
- Model metrics visualization
- Model comparison table
- Dataset statistics charts

## Components

- `LiveInference`: Upload and classify images
- `TrainingMonitor`: Track training progress
- `MetricsChart`: Visualize model performance
- `ModelComparison`: Compare multiple models
- `DatasetStats`: View dataset distribution
